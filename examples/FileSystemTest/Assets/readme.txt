SFilesystem Test App - Assets Directory
========================================

This folder contains test data files used by the FileSystemTest sample
to verify that sfs_get_assets_dir() and file-read operations work correctly
on all platforms (macOS, iOS, Android, Windows, Linux, WebAssembly).

Files:
  readme.txt           - this file
  test_data/sample.txt - plain text file for read/seek/tell/eof tests
  test_data/numbers.dat - binary-style data file for size tests
